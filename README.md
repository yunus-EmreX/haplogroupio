Haplogroup Explorer 🌍

An interactive 3D world map visualization tool for exploring major human Y-DNA haplogroups and their global distribution.

The project displays a rotating 3D Earth where users can select different haplogroups and see where they appear across the world. It is designed as a simple genetic-migration visualization tool that helps illustrate how paternal lineages spread geographically over time.

Interface Overview
3D Globe

The central component of the project is a rotating 3D Earth that highlights regions where the selected haplogroup appears.

Users can:

Rotate the globe

Hover regions

View frequency information

Compare global distributions

Haplogroup Selector

A sidebar lets users switch between haplogroups instantly.

Information Panel

Displays detailed information such as:

Estimated age of haplogroup

Origin region

Global carrier population

Highest regional frequencies
